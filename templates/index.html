How Accurate Is the Model?

These metrics are out-of-sample — calculated on the 20% of data the model never saw during training.
MAE — Average Error
--
On average, the model's price prediction is off by this much.
MSE — Squared Error
--
Penalises large errors more heavily. Lower is better.
RMSE — Error Value
--
The typical prediction error. Lower is better.
MAPE — Error in %
--
Average error as a percentage. Under 5% is considered good.
